# Chrome Web Store Listing — PhishBuster

## Short Description (132 chars max)
Automatically analyzes emails in Gmail for phishing indicators: suspicious URLs, spoofed senders, urgency tactics, and more.

## Detailed Description

PhishBuster protects you from phishing attacks by automatically analyzing every email you open in Gmail. It detects suspicious patterns and warns you with a clear banner before you fall for a scam.

**What it detects:**
- Suspicious URLs: IP addresses, URL shorteners, typosquatting domains, and link text that doesn't match the actual destination
- Spoofed senders: SPF, DKIM, and DMARC verification failures; mismatched From/Return-Path headers
- Social engineering: urgency language, threats, artificial deadlines, credential requests, and prize/reward bait
- Fake senders: free email providers impersonating brands, gibberish addresses, randomly generated domains
- Known threats: optional integration with VirusTotal, Google Safe Browsing, and PhishTank databases

**Features:**
- Instant analysis when you open any email in Gmail
- Color-coded risk banner (Low / Medium / High / Critical)
- Detailed alarm breakdown with severity levels
- Trusted senders & domains whitelist
- Multi-language support: English, Spanish, Catalan
- Daily analysis quota with easy top-ups
- Works in Chrome, Arc, Brave, Edge, and all Chromium browsers

**Privacy first:**
- Email content is analyzed in real-time and never stored
- All communication is encrypted via HTTPS
- No tracking, no cookies, no analytics
- Open-source analysis engine
